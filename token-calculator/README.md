# 🪙 Token 成本计算器

一个优雅的、基于浏览器的 LLM API 成本计算工具，支持实时计算、月度估算和模型价格管理。

## 功能特性

- **实时计算** — 输入 Token 数量即可秒出费用
- **预置热门模型** — DeepSeek v4 Flash/Pro、GPT-4o、Claude Sonnet 4
- **自定义模型** — 自由设定输入/输出单价
- **月度估算** — 输入每日请求量，自动计算日/月成本
- **动画数字过渡** — 数值变化时平滑动画
- **一键复制** — 生成格式化报告并复制到剪贴板
- **本地持久化** — 自动保存选中的模型和每日请求量
- **粒子背景** — 动态交互粒子效果
- **玻璃拟态 UI** — 毛玻璃卡片 + 深色主题
- **响应式设计** — 完美适配桌面和移动端

## 使用方式

直接在浏览器中打开 `index.html` 即可使用，无需任何服务器或依赖。
